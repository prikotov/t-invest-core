# T-Invest Skill

PHP console application for T-Invest API integration.

## Installation

```bash
composer require t-invest/skill
```

## Usage

```bash
./bin/skill
./bin/skill portfolio:positions
./bin/skill --help
```

## Available Commands

- `main` - Main command
- `portfolio:positions` - Get portfolio positions
- `help` - Display help

## Development

```bash
composer install
composer test
composer cs-check
composer stan
composer psalm
```
