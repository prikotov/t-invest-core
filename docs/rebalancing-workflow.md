# Workflow ребалансировки портфеля

## Обзор процесса

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         ЦИКЛ АНАЛИЗА                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. СБОР ДАННЫХ      2. АНАЛИЗ         3. РЕКОМЕНДАЦИИ                  │
│  ┌─────────┐        ┌─────────┐       ┌─────────────┐                  │
│  │Портфель │───────►│Метрики  │──────►│Что купить / │                  │
│  │ + Рынок │        │+ Сигналы│        │продать      │                  │
│  └─────────┘        └─────────┘       └─────────────┘                  │
│       │                  │                   │                          │
│       ▼                  ▼                   ▼                          │
│   API T-Invest      Тех.+Фунд.          Отчёт для                      │
│                     анализ              ручного исполнения              │
│                                                                         │
│                      ▼                                                  │
│              ВЫ ПРИНИМАЕТЕ РЕШЕНИЕ                                      │
│              И ИСПОЛНЯЕТЕ ВРУЧНУЮ                                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Принцип:** Советник только анализирует и рекомендует. Все решения и сделки — вы делаете сами.

## Справочник команд

| Команда | Описание |
|---------|----------|
| `portfolio:analyze` | Анализ портфеля, метрики, распределение |
| `portfolio:report` | Отчёт за период (week/month) |
| `portfolio:rebalance:plan` | Рекомендации по ребалансировке |
| `analyze:technical` | Технический анализ акции |
| `analyze:fundamental` | Фундаментальный анализ акции |
| `analyze:quick` | Быстрый обзор (тех + фунд) |
| `screen:stocks` | Скрининг акций по критериям |
